Customer churn is a critical challenge for subscription-based and service-driven businesses. This project predicts the likelihood that a customer will churn using demographic, behavioral, transactional, and product-level variables. The model is integrated into a full analytics pipeline with data processing, machine learning, and BI reporting.

ğŸ¯ Objectives

  • Analyze customer data to uncover churn drivers and behavioral patterns.
  • Train and evaluate multiple machine-learning models.
  • Select the best-performing model based on accuracy and stability.
  • Generate real churn probabilities for new customer datasets.
  • Deploy a Streamlit-based prediction app for non-technical business users.

ğŸ§  Machine Learning Pipeline

  1. Data preprocessing & cleaning (handling missing values, outliers, encoding)
  2. Feature engineering
  3. Train/test split
  4. Model training (Logistic Regression and Random Forest)
  5. Model selection (Random Forest was the best model)
  6. Final model export (.pkl)

ğŸ“ˆ Best Model: Random Forest Classifier

The Random Forest model delivered the highest predictive performance, achieving strong accuracy, stability, and robustness against overfitting. Its ability to capture non-linear relationships made it superior to simpler models. The final model outputs a churn probability score for each customer, which was later integrated into Power BI dashboards to support customer retention strategies.
