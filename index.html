üìå Project Overview

Customer churn is a major challenge for subscription-based or service-based businesses. The goal of this project is to create a predictive model that estimates the probability that a customer will churn, using demographic, behavioral, and usage-related variables.

üéØ Objectives

  • Analyze customer data to understand churn patterns.
  • Build and evaluate multiple machine-learning models.
  • Identify the best-performing model.
  • Generate churn probabilities for business teams.
  • Deploy a user-friendly prediction interface with Streamlit.

üß† Machine Learning Pipeline

  1. Data preprocessing & cleaning
  2. Feature engineering
  3. Train/test split
  4. Model training (Logistic Regression, Random Forest, XGBoost...)
  5. Model selection (Logistic Regression was best)
  6. Final model export (.pkl)

üìà Best Model: Logistic Regression

Logistic Regression achieved the most stable and interpretable performance, making it suitable for operational use and business interpretation. The model generates churn probabilities for each customer, which can be used in CRM systems and Power BI dashboards.
